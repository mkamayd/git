#Branch and merge locally
1. Create a new branch
```
git branch new-branch

```
2. Move to that branch
```
git checkout new-branch

```
3. Do some work 
```
git add *.*
git commit -m "some work"

```
4. Go back to main branch. 
```
git checkout master
git merge new-branch
\\done
git log

```
